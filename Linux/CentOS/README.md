# Linux
* Linux는 대표적인 오픈소스 운영체제.

## Linux History
* Linux는 UNIX를 모델로 만들었으며, 1991년 리누스 토발즈에 의해서 Kernel이 개발되었다. 그리고 GNU 프로젝트에 의해 Linux가 완성되었다.
* GNU 프로젝트는 리처드 스톨먼에 의해서 시작되었다. GNU's Not Unix의 약자로 UNIX와 호환이 되면서 더 강력한 운영체제를 만들고자 하는 프로젝트이다.
