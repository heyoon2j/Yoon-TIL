# Factory Pattern
객체 생성을 Factory Class가 대신 생성하는 패턴
* 종류로는 다음과 같다.
    1) Factory Method Pattern
    2) Abstract Factory Pattern
    > Factory Method는 어떤 객체를 생성할지에 대한 것(다형성)에 초점이 맞춰져있고, Abstract Factory는 서로 연관된 객체를 모아두는 것에 초점이 맞춰져있다.
