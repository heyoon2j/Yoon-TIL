# Code Convention
* https://www.python.org/dev/peps/pep-0008/


* Sorting
    * 반환 값이 필요없는 경우, list.sort()
    * 반환 값(iterable 객체)이 필요한 경우, sorted(iterable)

* Delete
    * 삭제 값을 반환할 필요가 없으면, del 문 사용
    * 반환해서 사용해야 하는 경우, pop() 사용  
    
* Tuple vs List
    * 사전에 요소의 수를 정확히 알고 있는 경우에는 Tuple
    아닌 경우에는 List
