
# Option
```
idle_in_transaction_session_timeout     # 트랜잭션 유휴 시간

pgaudit.log                             # 감사 로그
pgaudit.role                                ## 감사 로그 사용을 위한 Role

rds.force_ssl                           # SSL 연결 요구


ssl_max_protocol_version                # TLS Protocol version


password_encryption                     # 비밀번호 암호화

shared_preload_libraies                 # 라이브러리 extension : pgaudit, pg_tle 등 추가

pg_tle                                  # PostgreSQL 신뢰할 수 있는 언어 확장을 위한 오픈 소스 개발 키트

pgtle.enable_password_check             # 암호 확인 후 후크 사용 가능 여부
```
* 신뢰할 수 있는 언어는 특정 보안 속성을 가진 프로그래밍 또는 스크립팅 언어
