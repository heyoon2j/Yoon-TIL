## VRF
* 하나의 물리적인 라우터에서 여러 개의 가상 라우터(라우팅 테이블)가 존재하도록 하는 가상화 기술
* 3 Layer 동작
* VLAN vs VRF
    | 특징 | VLAN | VRF |
    |-----|---------|---------|
    | 계층 | Layer 2 | Layer 3 |
    | 기능 | LAN 분할 | 가상 라우터 생성 |
    | 격리 | Broadcast Domain 격리 | 라우팅 테이블 격리 |

    > VLAN / VXLAN / VRF 모두 가상 네트워크를 잘 사용하기 위해서, 같이 사용해야 되는 기술들이다.
