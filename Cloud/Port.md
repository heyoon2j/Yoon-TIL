# Port
* 운영체제 통신의 종단점.
* Port 번호는 다음 세가지로 나눌 수 있다.
    * 0 ~ 1023번: well-known port
    * 1024 ~ 49151번: registered port
    * 49152 ~ 65535번: dynamic port
